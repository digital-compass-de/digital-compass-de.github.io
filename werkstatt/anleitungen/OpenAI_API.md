# OpenAI API einrichten
#### Wer chatGPT in seinen Automatisierungen benutzen will, braucht einen API key von OpenAI und ein Guthaben, um die Nutzung zu bezahlen.

## URL
https://openai.com/api/

### Schritt 1
Über den Button "Registrieren" ein neues Konto erstellen, bzw. mit einem vorhanden chatGPT Account anmelden.
<br>
<img src="bilder/chatBot-erstellen-1.png" alt="Schritt 1 Telegram Bot" width="600" height="500">

### Schritt 2
Auf API key klicken (links im Menü) anschliessend auf "Create new secret key", rechts oben.
<br>
<img src="bilder/chatBot-erstellen-1.png" alt="Schritt 1 Telegram Bot" width="600" height="500">

### Schritt 3
Den Token kopieren und in die Automatisierungssoftware, OpenClaw, n8n, make beim AI-Agenten chatGPT einfügen.
